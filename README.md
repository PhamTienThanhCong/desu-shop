## install thư viện

- Tại Thư mục frontend:
    - install thư viện: `npm install --force`
    - run code: `npm start`

- Tại Thư mục admin:
    - install thư viện: `npm install --force`
    - run code: `npm start`

- Tại Thư mục api-chatbot:
    - install thư viện: `pip install requirements.txt`
    - run code: `python app.py`
